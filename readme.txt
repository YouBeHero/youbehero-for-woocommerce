With YouBeHero, you can enable donations directly on your WooCommerce store, empowering customers to contribute to organizations while shopping.

== Description ==

YouBeHero integrates seamlessly with WooCommerce to allow donations directly on your product, cart, checkout, and order confirmation pages. Encourage your customers or take charge as a shop owner to support charitable causes with each purchase.

**Features:**
- Add donation widgets on the product page, cart, checkout, and more.
- Flexible donation modes: Customers can round up totals or select fixed amounts.
- Shop owners can contribute a percentage or fixed amount per product.
- Customizable widget themes, positions, and styling.
- External credit-based donation system.
- Compatible with WooCommerce emails and order confirmations.

**How It Works:**
- Add a widget to display donation information and options.
- Donations can be paid by customers (as an extra fee) or the shop owner (predefined).
- Widgets are fully customizable, including themes, text, colors, and positions.

== Installation ==

1. Download and install the plugin:
   - Upload the `youbehero` folder to the `/wp-content/plugins/` directory.
   - Or install via the WordPress admin Plugins screen by searching for "YouBeHero".
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under WooCommerce > Settings > YouBeHero.

== Frequently Asked Questions ==

= How do I customize the widget's appearance? =
You can customize the widget's theme, colors, and positions via the plugin's settings in WooCommerce.

= Can I enable donations only on specific pages? =
Yes! You can control where the donation widget is displayed, such as product pages, the cart, checkout, and more.

= Does the plugin support email integration? =
Yes, the donation summary can appear in WooCommerce emails.

= How are donations processed? =
Donations are processed as part of the WooCommerce order, either as an additional fee (customer pays) or deducted from shop owner credits.
